# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. **Do NOT** open a public issue
2. Email the maintainer directly with details
3. Include steps to reproduce if possible
4. Allow reasonable time for a fix before disclosure

We take security seriously and will respond promptly to verified vulnerabilities.

## Security Best Practices

This app:
- Processes all face data on-device (no cloud uploads)
- Does not collect or transmit personal photos
- Uses standard Android permission model
- Does not require network access for core functionality
